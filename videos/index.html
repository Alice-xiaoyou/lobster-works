🎬 工作流程
· 点「渲染」 → HyperFrames 渲染视频（后台运行，约30-120秒）
· 点「预览」 → 查看 HTML 源码预览（浏览器新标签页）
· 渲染完成后 → 点「播放」查看视频文件
· 点「✅」 → 标记为已发布到视频号

🔧 自动化命令
· 渲染单个: cd videos/projects/项目名 && npx hyperframes render
· 批量渲染: ./render-all.sh
· 新项目: npx hyperframes init videos/projects/新项目名
