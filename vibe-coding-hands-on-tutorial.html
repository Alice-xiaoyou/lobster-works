Vibe Coding实战：不会写代码的人如何用AI做出第一个网站

英文YouTube 4000+教程 vs 中文互联网零。工具已经成熟，缺的是有人用中文告诉你该怎么上手。这是系统4天67篇内容中第一篇实操教程——不再讲"为什么"，直接教"怎么做"。

什么是Vibe Coding

Andrej Karpathy（OpenAI联合创始人）2025年创造这个词。核心：你不需要会编程，你只需要描述你想要什么，AI帮你写代码、调试、部署。

2026年中，两件关键的事已经发生：

第一，工具成熟了。 Replit Agent可以并行处理项目搭建、数据库配置、前端开发。Bolt.new主打专业设计，可以从Figma导入设计稿直接转成可运行的网站。

第二，竞争格局清晰了。 Replit做平台（免费入门、自带数据库、一键部署），Bolt做专业工具（设计系统支持、多模型路由）。不是竞争对手——是不同市场。

工具准备好了。缺的是有人用中文告诉你该怎么上手。

实战：10分钟用Replit做出第一个网站

以下步骤基于Replit前台界面实际观察 + 官方文档。标注了哪些已验证、哪些需要你在实操中自己试。

第1步：打开Replit，选"Agent"模式

注册后选Agent模式（不是Assistant——那是给有编程基础的人的）。你会看到一个类似聊天的界面，这就是你的"编程入口"。

第2步：用自然语言描述需求

关键是描述质量。不说"我要一个网站"，要说"我要一个个人博客网站，白底黑字，顶部有导航栏（首页、关于、文章），首页显示文章列表（标题+摘要+日期），文章详情页支持Markdown渲染"。不需要写一行代码——但需要把需求讲清楚。这个技能比学编程简单100倍。

第3步：看Agent工作，实时反馈

Agent会搭建结构→配置依赖→写前端→写后端→配置数据库→部署URL。不满意就在聊天里说"导航栏改成黑色的"，Agent会改。

第4步：一键部署上线

不需要服务器、域名、Docker——点一下就有公网URL。免费版够验证想法，付费$25/月起。

Bolt.new：如果你想要更好的设计感

Replit适合"快速验证"，Bolt适合"做得好看"。可以从Figma导入设计稿、自动选最优AI模型。不是谁更好——是谁更适合你现在要做的事。

为什么现在是你的窗口期

三个条件叠加创造天然壁垒：真用过工具 + 中文表达能力好 + 持续更新。同时满足这三个条件的人≈零。谁先入场，谁就定义这个赛道的标准。

今天就可以开始的三个动作

1. 注册Replit（5分钟）→ Agent模式 → 输入需求 → 拿到第一个URL

2. 录屏或截图。你的第一步本身就是最好的教学内容

3. 分享你的第一个作品。评论区的问题就是下一篇教程选题

中文互联网最大的AI内容蓝海，不在"怎么做AI"、不在"AI会不会取代你"——在"不会写代码的人怎么利用AI做东西"。这篇文章就是这张蓝海的第一个坐标。
